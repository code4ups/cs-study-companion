##

    Τα δίκτυα δημιουργούνται για συγκεκριμένους σκοπούς και έχουν χαρακτηριστικά που τα κάνουν κατάλληλα για διαφορετικές χρήσεις.

    ### Κύριοι Σκοποί των Δικτύων

    #### 1. Κοινή Χρήση Πόρων (Resource Sharing)
    - **Υλικό (Hardware)**: εκτυπωτές, σαρωτές, αποθηκευτικά μέσα
    - **Λογισμικό (Software)**: εφαρμογές, βάσεις δεδομένων, λειτουργικά συστήματα
    - **Δεδομένα**: αρχεία, έγγραφα, πληροφορίες

    #### 2. Επικοινωνία (Communication)
    - Email και instant messaging
    - Video conferencing και VoIP
    - Collaborative tools

    #### 3. Πρόσβαση σε Υπηρεσίες
    - Internet browsing
    - Cloud services
    - Online applications

    ### Βασικά Χαρακτηριστικά

    <KeyTerm lang="el">
        **Bandwidth (Εύρος Ζώνης)**: Η μέγιστη ποσότητα δεδομένων που μπορεί να μεταδοθεί μέσω μιας σύνδεσης σε συγκεκριμένο χρονικό διάστημα, μετριέται σε bits per second (bps).
    </KeyTerm>

    #### Απόδοση (Performance)
    - **Bandwidth**: θεωρητική ικανότητα
    - **Throughput**: πραγματική μετάδοση
    - **Latency**: καθυστέρηση

    #### Αξιοπιστία (Reliability)
    - Fault tolerance
    - Error detection και correction
    - Redundancy
