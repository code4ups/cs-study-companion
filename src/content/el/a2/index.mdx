# A2 – Δίκτυα

<GuidingQuestion lang="el">
    Πώς είναι δυνατόν οι υπολογιστές και οι συσκευές να επικοινωνούν μεταξύ τους μέσω δικτύων σε ολόκληρο τον κόσμο;
</GuidingQuestion>

Οι περισσότεροι χρήστες, όταν χρησιμοποιούν υπολογιστές, σπάνια σκέφτονται την τεχνολογία που τους επιτρέπει να συνδέονται με άλλους ή να έχουν πρόσβαση σε πληροφορίες στο διαδίκτυο. Τα δίκτυα θεωρούνται δεδομένα: μια συσκευή συνδέεται σε ένα Wi-Fi και «απλώς λειτουργεί». Ωστόσο, πίσω από αυτήν την απλότητα κρύβεται μια σύνθετη δομή κανόνων, πρωτοκόλλων και υλικού που καθιστούν δυνατή την επικοινωνία.

Ως επιστήμονας της πληροφορικής, ο ρόλος σου είναι να διερευνήσεις πώς λειτουργούν αυτά τα συστήματα.
**Τι ακριβώς είναι μια διεύθυνση IP και γιατί είναι απαραίτητη;**
**Πώς μπορεί να παρακολουθείται η διαδικτυακή κίνηση και ποιοι περιορισμοί υπάρχουν για την προστασία της ιδιωτικότητας;**
**Με ποιον τρόπο τα ονόματα τομέων μεταφράζονται σε διευθύνσεις IP;**
**Ποια προβλήματα δημιουργεί η έλλειψη διευθύνσεων IP και πώς εξασφαλίζουν τα πρωτόκολλα ότι τα δεδομένα παραδίδονται αξιόπιστα σε όλο τον κόσμο;**

Είναι εντυπωσιακό το γεγονός ότι ένα αίτημα μπορεί να ταξιδέψει σε ολόκληρες ηπείρους και να επιστρέψει με απάντηση σε λιγότερο από ένα δευτερόλεπτο. Μελετώντας αυτές τις διαδικασίες, θα εμβαθύνεις στην κατανόηση των δικτύων, του υλικού και των πρωτοκόλλων που στηρίζουν την παγκόσμια επικοινωνία.
