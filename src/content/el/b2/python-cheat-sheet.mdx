# Cheat Sheet Python - Βοήθημα

---

## **Οι βασικοί τύποι οντοτήτων της Python**

| **Τύπος** | **Περιγραφή** |
| --------- | ------------- |
| **int**   | Ακέραιος π.χ. 1, 2, -1, 1000, κλπ. |
| **float** | Πραγματικός αριθμός π.χ. 5.5, 3.14, κλπ. |
| **str**   | Συμβολοσειρά π.χ. 'Python', 'ΛΕΣΒΟΣ', κλπ. |
| **bool**  | Λογικός τύπος με επιτρεπόμενες τιμές `True` ή `False` |
| **tuple** | Πλειάδα π.χ. (12, 'Νίκος', True, 3.14) |
| **set**   | Σύνολο π.χ. {12,5,6,5,'Τέλος'} |
| **list**  | Λίστα π.χ. ['Python', 3.11, 'C++', 11, 'Java', 8.0] |
| **dict**  | Λεξικό π.χ. &#123;'Python'&#58; 3.11, 'C++'&#58; 11, 'Java'&#58; 8.0&#125; |
| **range** | Περιοχή τιμών π.χ. range(5,15,2) |

---

## **Τελεστές**

Οι πιο συχνά χρησιμοποιούμενοι τελεστές κατά σειρά προτεραιότητας είναι οι εξής:

* `()` (πράξεις μέσα σε παρενθέσεις)
* `**` (ύψωση σε δύναμη)
* `*`, `/`, `//`, `%` (πολλαπλασιασμός, διαίρεση, υπόλοιπο ακέραιας διαίρεσης)
* `+`, `-` (πρόσθεση και αφαίρεση)
* `==`, `!=`, `>`, `>=`, `<`, `<=` (λογικοί τελεστές: ίσο, διάφορο, μεγαλύτερο, μεγαλύτερο ή ίσο, μικρότερο, μικρότερο ή ίσο)
* `not` (λογικό ΟΧΙ)
* `and` (λογικό ΚΑΙ)
* `or` (λογικό Ή)
* `=` (τελεστής ανάθεσης)

## **Είσοδος/Έξοδος στην κονσόλα**

**Έξοδος στην οθόνη:**

```python
print(παρ1, παρ2, ..., παρΝ)
```

Η συνάρτηση `print` εμφανίζει τις τιμές των παραστάσεων παρ1, παρ2, ....

**Είσοδος από το πληκτρολόγιο:**

```python
m = input('Μήνυμα')
```

Η συνάρτηση `input` εμφανίζει το μήνυμα και περιμένει τον χρήστη να πληκτρολογήσει κάτι. Η συνάρτηση επιστρέφει μία συμβολοσειρά με τους χαρακτήρες που πληκτρολόγησε ο χρήστης.

---

#### Παράδειγμα ολοκληρωμένου προγράμματος

```python
# Σχόλια μπορούν να μπουν σε οποιοδήποτε σημείο του προγράμματος
import math
ar = float(input('Δώσε έναν αριθμό: '))
if ar > 0:
    print(math.sqrt(ar))  # υπολογίζει την τετραγωνική ρίζα
else:
    print('Τέλος.')
```

📌 Η εντολή `import` εισάγει κάποιο άρθρωμα, ώστε να μπορούμε να χρησιμοποιήσουμε τους πόρους του.

## **Διάφορες χρήσιμες συναρτήσεις**

| **Συνάρτηση** | **Περιγραφή** |
| ------------- | ------------- |
| `id(Οντότητα)` | Επιστρέφει τη διεύθυνση της Οντότητας ως ακέραιο. |
| `type(Οντότητα)` | Επιστρέφει ως συμβολοσειρά τον τύπο της Οντότητας. |
| `int(Οντότητα)` | Μετατρέπει μια οντότητα σε ακέραιο αριθμό (αν είναι δυνατή η μετατροπή). |
| `float(Οντότητα)` | Μετατρέπει μια οντότητα σε πραγματικό αριθμό (αν είναι δυνατή η μετατροπή). |
| `str(Οντότητα)` | Μετατρέπει μια οντότητα σε συμβολοσειρά. Χωρίς όρισμα επιστρέφει μια κενή συμβολοσειρά. |
| `list(Οντότητα)` | Μετατρέπει μια επαναληπτέα οντότητα σε λίστα. Χωρίς όρισμα επιστρέφει μια κενή λίστα. |
| `tuple(Οντότητα)` | Μετατρέπει μια επαναληπτέα οντότητα σε πλειάδα. Χωρίς όρισμα επιστρέφει μια κενή πλειάδα. |
| `set(Οντότητα)` | Μετατρέπει μια επαναληπτέα οντότητα σε σύνολο. Χωρίς όρισμα επιστρέφει ένα κενό σύνολο. |
| `range(Αρχή, Τέλος, Βήμα)` | Επιστρέφει μια οντότητα περιοχής, η οποία περιλαμβάνει αριθμούς από την αρχική μέχρι την τιμή Τέλος (χωρίς να περιλαμβάνεται το Τέλος), με βήμα αύξησης ή μείωσης την τιμή Βήμα (προαιρετικά). |
| `max(Οντότητα)` | Επιστρέφει ως τιμή το στοιχείο με τη μεγαλύτερη τιμή στην οντότητα (Συμβολοσειρές, Πλειάδες, Λίστες, Σύνολα ή Λεξικά). |
| `min(Οντότητα)` | Επιστρέφει ως τιμή το στοιχείο με τη μικρότερη τιμή στην οντότητα. |
| `len(Οντότητα)` | Επιστρέφει ως τιμή το πλήθος των στοιχείων μιας σύνθετης οντότητας. |
