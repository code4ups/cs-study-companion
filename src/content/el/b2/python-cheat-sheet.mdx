## **Οι βασικοί τύποι οντοτήτων της Python**

| **Τύπος** | **Περιγραφή** |
| --------- | ------------- |
| **int**   | Ακέραιος π.χ. 1, 2, -1, 1000, κλπ. |
| **float** | Πραγματικός αριθμός π.χ. 5.5, 3.14, κλπ. |
| **str**   | Συμβολοσειρά π.χ. 'Python', 'ΛΕΣΒΟΣ', κλπ. |
| **bool**  | Λογικός τύπος με επιτρεπόμενες τιμές `True` ή `False` |
| **tuple** | Πλειάδα π.χ. (12, 'Νίκος', True, 3.14) |
| **set**   | Σύνολο π.χ. {12,5,6,5,'Τέλος'} |
| **list**  | Λίστα π.χ. ['Python', 3.11, 'C++', 11, 'Java', 8.0] |
| **dict**  | Λεξικό π.χ. &#123;'Python'&#58; 3.11, 'C++'&#58; 11, 'Java'&#58; 8.0&#125; |
| **range** | Περιοχή τιμών π.χ. range(5,15,2) |

## **Τελεστές**

Οι πιο συχνά χρησιμοποιούμενοι τελεστές κατά σειρά προτεραιότητας είναι οι εξής:

* `()` (πράξεις μέσα σε παρενθέσεις)
* `**` (ύψωση σε δύναμη)
* `*`, `/`, `//`, `%` (πολλαπλασιασμός, διαίρεση, υπόλοιπο ακέραιας διαίρεσης)
* `+`, `-` (πρόσθεση και αφαίρεση)
* `==`, `!=`, `>`, `>=`, `<`, `<=` (λογικοί τελεστές: ίσο, διάφορο, μεγαλύτερο, μεγαλύτερο ή ίσο, μικρότερο, μικρότερο ή ίσο)
* `not` (λογικό ΟΧΙ)
* `and` (λογικό ΚΑΙ)
* `or` (λογικό Ή)
* `=` (τελεστής ανάθεσης)

## **Είσοδος/Έξοδος στην κονσόλα**

**Έξοδος στην οθόνη:**

```python
print(παρ1, παρ2, ..., παρΝ)
```

Η συνάρτηση `print` εμφανίζει τις τιμές των παραστάσεων παρ1, παρ2, ....

**Είσοδος από το πληκτρολόγιο:**

```python
m = input('Μήνυμα')
```

Η συνάρτηση `input` εμφανίζει το μήνυμα και περιμένει τον χρήστη να πληκτρολογήσει κάτι. Η συνάρτηση επιστρέφει μία συμβολοσειρά με τους χαρακτήρες που πληκτρολόγησε ο χρήστης.

---

#### Παράδειγμα ολοκληρωμένου προγράμματος

```python
# Σχόλια μπορούν να μπουν σε οποιοδήποτε σημείο του προγράμματος
import math
ar = float(input('Δώσε έναν αριθμό: '))
if ar > 0:
    print(math.sqrt(ar))  # υπολογίζει την τετραγωνική ρίζα
else:
    print('Τέλος.')
```

📌 Η εντολή `import` εισάγει κάποιο άρθρωμα, ώστε να μπορούμε να χρησιμοποιήσουμε τους πόρους του.

## **Διάφορες χρήσιμες συναρτήσεις**

| **Συνάρτηση** | **Περιγραφή** |
| ------------- | ------------- |
| `id(Οντότητα)` | Επιστρέφει τη διεύθυνση της Οντότητας ως ακέραιο. |
| `type(Οντότητα)` | Επιστρέφει ως συμβολοσειρά τον τύπο της Οντότητας. |
| `int(Οντότητα)` | Μετατρέπει μια οντότητα σε ακέραιο αριθμό (αν είναι δυνατή η μετατροπή). |
| `float(Οντότητα)` | Μετατρέπει μια οντότητα σε πραγματικό αριθμό (αν είναι δυνατή η μετατροπή). |
| `str(Οντότητα)` | Μετατρέπει μια οντότητα σε συμβολοσειρά. Χωρίς όρισμα επιστρέφει μια κενή συμβολοσειρά. |
| `list(Οντότητα)` | Μετατρέπει μια επαναληπτέα οντότητα σε λίστα. Χωρίς όρισμα επιστρέφει μια κενή λίστα. |
| `tuple(Οντότητα)` | Μετατρέπει μια επαναληπτέα οντότητα σε πλειάδα. Χωρίς όρισμα επιστρέφει μια κενή πλειάδα. |
| `set(Οντότητα)` | Μετατρέπει μια επαναληπτέα οντότητα σε σύνολο. Χωρίς όρισμα επιστρέφει ένα κενό σύνολο. |
| `range(Αρχή, Τέλος, Βήμα)` | Επιστρέφει μια οντότητα περιοχής, η οποία περιλαμβάνει αριθμούς από την αρχική μέχρι την τιμή Τέλος (χωρίς να περιλαμβάνεται το Τέλος), με βήμα αύξησης ή μείωσης την τιμή Βήμα (προαιρετικά). |
| `max(Οντότητα)` | Επιστρέφει ως τιμή το στοιχείο με τη μεγαλύτερη τιμή στην οντότητα (Συμβολοσειρές, Πλειάδες, Λίστες, Σύνολα ή Λεξικά). |
| `min(Οντότητα)` | Επιστρέφει ως τιμή το στοιχείο με τη μικρότερη τιμή στην οντότητα. |
| `len(Οντότητα)` | Επιστρέφει ως τιμή το πλήθος των στοιχείων μιας σύνθετης οντότητας. |

## **Δομές ελέγχου**

### **Εντολή `if`**

```python
if συνθήκη:
    μπλοκ προτάσεων για αληθή συνθήκη
```

```python
if συνθήκη:
    μπλοκ προτάσεων για αληθή συνθήκη
else:
    μπλοκ προτάσεων για ψευδή συνθήκη
```

#### Παραδείγματα:

```python
if x < y:
    print('x:', x)
```

```python
if x < y:
    print('======')
    print(x)
else:
    print('######')
    print(y)
```

---

### **Εντολή `if-elif`**

```python
if συνθήκη1:
    μπλοκ προτάσεων για αληθή συνθήκη1
elif συνθήκη2:
    μπλοκ προτάσεων για αληθή συνθήκη2
elif συνθήκη3:
    μπλοκ προτάσεων για αληθή συνθήκη3
else:
    μπλοκ προτάσεων για ψευδείς συνθήκες
```

#### Παραδείγματα:

```python
if a == 5:
    print('a=5')
elif a == 8:
    print('a=8')
elif a > 10:
    print('Μεγάλος')
else:
    print('ΤΕΛΟΣ')
```

📌 **Όταν όλες οι συνθήκες είναι ψευδείς, τότε εκτελείται το else τμήμα, αυτό είναι προαιρετικό.**

## **Δομές επανάληψης**

### **Βρόχος `while`**

```python
while συνθήκη:
    μπλοκ προτάσεων
```

---
## **Επεξηγηματικό Πλαίσιο (μπλε φόντο)**

📌 Στον βρόχο `while` ο έλεγχος της συνθήκης γίνεται στην αρχή. Το μπλοκ προτάσεων εκτελείται όσο η συνθήκη είναι αληθής.


#### Παράδειγμα:

```python
while x < 100:
    print(x)
    x = x + 1
```

---

### **Βρόχος `for`**

```python
for m in Οντότητα:
    πρόταση
```

📌 Όπου **Οντότητα** είναι οποιαδήποτε επαναληπτέα οντότητα της Python. Στη μεταβλητή **m** ανατίθενται διαδοχικά τα στοιχεία της επαναληπτέας οντότητας.

#### Παράδειγμα:

```python
for i in range(10):
    print(i)
```

📌 Εμφανίζει τους αριθμούς από το 0 μέχρι το 9.

---

```python
s = 0
for i in (1, 5, 7, 9):
    s = s + i
    print(s)
```

📌 Εμφανίζει έναν προς έναν τους αθροιστικούς ενδιάμεσους πίνακες καθώς και το άθροισμα.

## **Εντολές `break` και `continue`**

#### Παράδειγμα:

```python
while x < 100:
    if x == 38:
        break
    elif x % 5 == 0:
        continue
    print(x)
    x = x + 2
```

📌 Η εντολή **break** διακόπτει την εκτέλεση ενός βρόχου `while` ή `for` και μεταφέρει τον έλεγχο του προγράμματος αμέσως **μετά** από τη δομή επανάληψης.

📌 Η εντολή **continue** επιβάλλει την εκτέλεση **της επόμενης επανάληψης** του βρόχου, παραλείποντας τις ενδιάμεσες εντολές.

---

## **Επεξηγηματικό Πλαίσιο (μπλε φόντο)**

📌 Οι δομές ελέγχου, καθώς και οι δομές επανάληψης, εκτός από λογικές συνθήκες ελέγχουν και την τιμή παραστάσεων. Μια παράσταση στην Python θεωρείται **ψευδής** όταν έχει τιμή `0`, κενό, ή `None`, και **αληθής** σε κάθε άλλη περίπτωση.

## **Συμβουλές & Σημειώσεις**

📍 **Επαναληπτές** οντότητες είναι οι συμβολοσειρές, οι πλειάδες, οι λίστες, τα σύνολα, τα λεξικά, οι περιοχές τιμών και οι μεταβλητές.

📍 Οι λίστες, τα σύνολα και τα λεξικά είναι **μεταβαλλόμενες** οντότητες. Δηλαδή, μπορούμε να αλλάξουμε κάποιο από τα στοιχεία τους.

📍 Αντίθετα, οι **συμβολοσειρές**, οι **πλειάδες** και η **περιοχή τιμών** είναι **μη μεταβαλλόμενες**. Η αναφορά τους γίνεται με δείκτη.

📍 Η **πλειάδα** μπορεί να περιέχει **μεταβλητές**, **λίστες** και **οι πλειάδες να είναι εμφωλευμένες**. Μπορούμε να αλλάξουμε κάποια τιμή τους **μέσω των μεταβλητών**.

📍 Οι λίστες, οι πλειάδες, τα σύνολα και τα λεξικά **υποστηρίζουν αριθμητικούς τελεστές**. Η χρήση αριθμητικών τελεστών στις επαναληπτικές οντότητες γίνεται με χρήση αριθμοδείκτη.
