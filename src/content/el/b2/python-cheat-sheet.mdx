## **Οι βασικοί τύποι οντοτήτων της Python**

| **Τύπος** | **Περιγραφή** |

## **Τελεστές**

Οι πιο συχνά χρησιμοποιούμενοι τελεστές κατά σειρά προτεραιότητας είναι οι εξής:

* `()` (πράξεις μέσα σε παρενθέσεις)
* `**` (ύψωση σε δύναμη)
* `*`, `/`, `//`, `%` (πολλαπλασιασμός, διαίρεση, υπόλοιπο ακέραιας διαίρεσης)
* `+`, `-` (πρόσθεση και αφαίρεση)
* `==`, `!=`, `>`, `>=`, `<`, `<=` (λογικοί τελεστές: ίσο, διάφορο, μεγαλύτερο, μεγαλύτερο ή ίσο, μικρότερο, μικρότερο ή ίσο)
* `not` (λογικό ΟΧΙ)
* `and` (λογικό ΚΑΙ)
* `or` (λογικό Ή)
* `=` (τελεστής ανάθεσης)

## **Είσοδος/Έξοδος στην κονσόλα**

**Έξοδος στην οθόνη:**

```python
print(παρ1, παρ2, ..., παρΝ)
```

Η συνάρτηση `print` εμφανίζει τις τιμές των παραστάσεων παρ1, παρ2, ....

**Είσοδος από το πληκτρολόγιο:**

```python
m = input('Μήνυμα')
```

Η συνάρτηση `input` εμφανίζει το μήνυμα και περιμένει τον χρήστη να πληκτρολογήσει κάτι. Η συνάρτηση επιστρέφει μία συμβολοσειρά με τους χαρακτήρες που πληκτρολόγησε ο χρήστης.

---

#### Παράδειγμα ολοκληρωμένου προγράμματος

```python
# Σχόλια μπορούν να μπουν σε οποιοδήποτε σημείο του προγράμματος
import math
ar = float(input('Δώσε έναν αριθμό: '))
if ar > 0:
    print(math.sqrt(ar))  # υπολογίζει την τετραγωνική ρίζα
else:
    print('Τέλος.')
```

📌 Η εντολή `import` εισάγει κάποιο άρθρωμα, ώστε να μπορούμε να χρησιμοποιήσουμε τους πόρους του.

### **Εντολή `if`**

```python

    μπλοκ προτάσεων για αληθή συνθήκη
```

```python
if x < y:
    print('======')
    print(x)

### **Εντολή `if-elif`**

```python
if συνθήκη1:
    μπλοκ προτάσεων για αληθή συνθήκη1
elif συνθήκη2:
    μπλοκ προτάσεων για αληθή συνθήκη2
elif συνθήκη3:
    μπλοκ προτάσεων για αληθή συνθήκη3
else:
    μπλοκ προτάσεων για ψευδείς συνθήκες

#### Παραδείγματα:

```python
if a == 5:

elif a == 8:
    print('a=8')
elif a > 10:

else:
    print('ΤΕΛΟΣ')
```

📌 **Όταν όλες οι συνθήκες είναι ψευδείς, τότε εκτελείται το else τμήμα, αυτό είναι προαιρετικό.**

## **Επεξηγηματικό Πλαίσιο (μπλε φόντο)**

### **Βρόχος `for`**

```python
for m in Οντότητα:
    πρόταση
```

📌 Όπου **Οντότητα** είναι οποιαδήποτε επαναληπτέα οντότητα της Python. Στη μεταβλητή **m** ανατίθενται διαδοχικά τα στοιχεία της επαναληπτέας οντότητας.

#### Παράδειγμα:

```python
for i in range(10):
    print(i)
```

📌 Εμφανίζει τους αριθμούς από το 0 μέχρι το 9.

---

```python
s = 0
for i in (1, 5, 7, 9):
    s = s + i
    print(s)
```

## **Εντολές `break` και `continue`**

#### Παράδειγμα:

```python
while x < 100:
    if x == 38:
        break
    elif x % 5 == 0:
        continue
    print(x)
    x = x + 2
```

📌 Η εντολή **break** διακόπτει την εκτέλεση ενός βρόχου `while` ή `for` και μεταφέρει τον έλεγχο του προγράμματος αμέσως **μετά** από τη δομή επανάληψης.

📌 Η εντολή **continue** επιβάλλει την εκτέλεση **της επόμενης επανάληψης** του βρόχου, παραλείποντας τις ενδιάμεσες εντολές.

## **Συμβουλές & Σημειώσεις**

📍 **Επαναληπτές** οντότητες είναι οι συμβολοσειρές, οι πλειάδες, οι λίστες, τα σύνολα, τα λεξικά, οι περιοχές τιμών και οι μεταβλητές.

📍 Οι λίστες, τα σύνολα και τα λεξικά είναι **μεταβαλλόμενες** οντότητες. Δηλαδή, μπορούμε να αλλάξουμε κάποιο από τα στοιχεία τους.

📍 Αντίθετα, οι **συμβολοσειρές**, οι **πλειάδες** και η **περιοχή τιμών** είναι **μη μεταβαλλόμενες**. Η αναφορά τους γίνεται με δείκτη.

📍 Η **πλειάδα** μπορεί να περιέχει **μεταβλητές**, **λίστες** και **οι πλειάδες να είναι εμφωλευμένες**. Μπορούμε να αλλάξουμε κάποια τιμή τους **μέσω των μεταβλητών**.

📍 Οι λίστες, οι πλειάδες, τα σύνολα και τα λεξικά **υποστηρίζουν αριθμητικούς τελεστές**. Η χρήση αριθμητικών τελεστών στις επαναληπτικές οντότητες γίνεται με χρήση αριθμοδείκτη.

## **Τμηματοποίηση**

Η **τμηματοποίηση** μιας ακολουθιακής οντότητας (αο), γίνεται με μια παράσταση της μορφής:

```python
αο[Αρχή:Τέλος:Βήμα]
```

* Η **Αρχή** είναι αριθμοδείκτης που προσδιορίζει την αρχή,
* το **Τέλος** είναι το τέλος του τμήματος,
* και το **Βήμα** καθορίζει πόσα βήματα επιλογής να γίνει κάθε φορά.

Αν το **Βήμα** είναι **αρνητικό**, η φορά διάσχισης και επιλογής είναι **αντίστροφη**, δηλαδή από τα δεξιά προς τα αριστερά.
Όταν παραληφθούν:

* η **Αρχή** θεωρείται το πρώτο στοιχείο,
* το **Τέλος** θεωρείται το τελευταίο στοιχείο,
* και το **Βήμα** θεωρείται 1.

### Παραδείγματα

```python
s = '123456789ΑΒΓΔ'
```

| Παράσταση    | Περιγραφή                                                                               |
| ------------ | --------------------------------------------------------------------------------------- |
| `s[2:8:1]`   | Επιστρέφει το τμήμα της συμβολοσειράς από τον 2ο μέχρι και τον 7ο χαρακτήρα: `'345678'` |
| `s[2:8:2]`   | από τον 2ο μέχρι και τον 7ο χαρακτήρα ανά 2: `'357'`                                    |
| `s[2::2]`    | από τον 2ο και μετά μέχρι το τέλος, ανά 2: `'3579Α'`                                    |
| `s[::3]`     | από την αρχή μέχρι το τέλος, ανά 3: `'147Α'`                                            |
| `s[::-1]`    | από το τέλος προς την αρχή (αντιστροφή): `'ΔΓΒΑ987654321'`                              |
| `s[3:-1:-1]` | δεν εμφανίζεται τίποτα: `' '` (λάθος κατεύθυνση επιλογής)                               |
| `s[-3::-1]`  | από τον τρίτο από το τέλος μέχρι την αρχή, ανά -1: `'98765'`                            |

## **Συναρτήσεις**

Μια συνάρτηση μπορεί να επιστρέφει **μία ή καμία τιμή**.
Μπορούμε να μεταβιβάσουμε τιμές σε μια συνάρτηση μέσω των παραμέτρων της.

### Γενική μορφή συνάρτησης

```python
def ΌνομαΣυνάρτησης(p1, p2, ..., pN):
    μπλοκ προτάσεων
    return τιμή
```

📌 Η εντολή `return` καθορίζει την τιμή που επιστρέφει η συνάρτηση.

---

### Παραδείγματα

```python
def mo(a, b):
    mm = (a + b) / 2
    return mm
```

📌 Η συνάρτηση διαθέτει δύο παραμέτρους και επιστρέφει ως τιμή τον **μέσο όρο**.

```python
def doit():
    for i in range(10):
        print('Python')
```

📌 Η συνάρτηση δεν δέχεται παραμέτρους και δεν επιστρέφει τιμή. Απλώς εκτελεί εντολές. Εμφανίζει τη λέξη *Python* 10 φορές.

---

### Κλήση Συναρτήσεων

#### Κυρίως πρόγραμμα:

```python
k = 5
m = 10
mesos = mo(k, m)         # Κλήση της mo() με ορίσματα k και m
print('Μέσος όρος:', mesos)

mesos = mo(15, 6)         # Κλήση της mo() με ορίσματα 15 και 6
print('Μέσος όρος:', mesos)
```

📌 Όταν καλείται μια συνάρτηση με ορίσματα μεταβλητού τύπου (λίστες, σύνολα, λεξικά), οι **τιμές των παραμέτρων δεν αντιγράφονται**, αλλά αντιγράφεται η **αναφορά** στη θέση μνήμης της αρχικής οντότητας. Αυτό σημαίνει ότι η συνάρτηση **μπορεί να μεταβάλει** την αρχική οντότητα.

📌 Αν όμως καλείται συνάρτηση με απλές τιμές (ακέραιοι, πραγματικοί, λογικές τιμές, συμβολοσειρές), τότε **δεν μπορεί** να τροποποιήσει την αρχική οντότητα, αφού αυτές είναι **αμετάβλητοι τύποι**.

📌 Η συνάρτηση μπορεί να επιστρέφει **μία τιμή**, και στη συνέχεια αυτή να μπορεί να χρησιμοποιηθεί σε εκφράσεις, αναθέσεις, εμφάνιση στην οθόνη κ.λπ.
Επίσης, μπορεί να επιστρέφει **περισσότερες από μία τιμές**, που αντιστοιχούν σε **πλειάδες** ή **λίστες**.

## **Συμβολοσειρές**

Οι **κυριολεκτικές συμβολοσειρές (str)** στην Python είναι χαρακτήρες μέσα σε **απλά (')** ή **διπλά (")** εισαγωγικά.
Μπορούμε να εσωκλείσουμε μια συμβολοσειρά μέσα σε **τριπλά απλά (''')** ή **τριπλά διπλά (""") εισαγωγικά**, ώστε να περιέχει πολλαπλές γραμμές.

Στην περίπτωση αυτή οι αλλαγές γραμμής, τα εισαγωγικά κ.λπ. περιλαμβάνονται κανονικά.
Τα **διπλά εισαγωγικά** και τα **μονά εισαγωγικά** μπορούν να περιλαμβάνονται ως **κανονικοί χαρακτήρες**.

Οι συμβολοσειρές είναι **μη μεταβλητός τύπος**.

---

## Πίνακας: Μέθοδοι που εφαρμόζονται σε συμβολοσειρές

| **Μέθοδος** | **Περιγραφή** |
| ----------- | ------------- |
| `count(str)` | Επιστρέφει το πλήθος εμφανίσεων της συμβολοσειράς `str` μέσα στη συμβολοσειρά. |
| `encode()` | Επιστρέφει μια νέα έκδοση με την **κωδικοποιημένη** σε έκδοση utf-8 συμβολοσειρά. |
| `find(str)` | Επιστρέφει τον **πρώτο δείκτη** που εντοπίζει τη συμβολοσειρά `str` μέσα στην αρχική. Αν δεν τη βρει, επιστρέφει `-1`. |
| `format()` | Μετατρέπει τιμές σε συμβολοσειρές σύμφωνα με συγκεκριμένο πρότυπο. |
| `join(obj)` | Μετατρέπει στοιχεία ενός επαναληπτικού αντικειμένου σε συμβολοσειρά, με **ενδιάμεσο** χαρακτήρα αυτόν που καλεί τη μέθοδο. |
| `split()` | Χωρίζει μια συμβολοσειρά σε λίστα υποσυμβολοσειρών. |
| `lower()` | Επιστρέφει νέα έκδοση της συμβολοσειράς αφού μετατρέψει όλους τους χαρακτήρες της σε **πεζούς**. |
| `partition(str)` | Επιστρέφει μια πλειάδα με στοιχεία τρία τμήματα της συμβολοσειράς, ανάλογα με το που εντοπίζεται η πρώτη φορά η συμβολοσειρά `str`. |
| `replace(str1, str2)` | Όπου η συμβολοσειρά `str1` υπάρχει στη συμβολοσειρά `s`, αυτή αντικαθίσταται από τη `str2`. |
| `split(str)` | Χωρίζει τη συμβολοσειρά σε τμήματα, χρησιμοποιώντας ως διαχωριστικό τη συμβολοσειρά `str`. Επιστρέφει μια λίστα με τα διακριτά τμήματα. |
| `splitlines()` | Χωρίζει τη συμβολοσειρά σε τμήματα, χρησιμοποιώντας ως διαχωριστικά χαρακτήρες αλλαγής γραμμής. |
| `strip()` | Επιστρέφει νέα έκδοση της συμβολοσειράς αφαιρώντας τα κενά από την αρχή και το τέλος της. |
| `upper()` | Επιστρέφει νέα έκδοση της συμβολοσειράς αφού μετατρέπει όλους τους χαρακτήρες της σε κεφαλαίους. |

---

### **Παραδείγματα**

```python
s = '12-567-ΑΒΓΔ'
s.lower()         → '12-567-αβγδ'
s.split('-')      → ['12', '567', 'ΑΒΓΔ']
s.count('-')      → 2
s.replace('-', '#') → '12#567#ΑΒΓΔ'
s.partition('567')  → ('12-', '567', '-ΑΒΓΔ')
```

📌 Με τη μέθοδο `format()`, τα σύμβολα κράτησης θέσης `{}` σε μια συμβολοσειρά αντικαθίστανται από τις τιμές μεταβλητών.
```python
a = 1; b = 2; f = 'β'
print('{0}={1}  {2}={3}'.format(a,1,b,f))  # → 'A=1  B=2  Δ=β'```

📌 Η μέθοδος `format()` μπορεί να χρησιμοποιηθεί και για μορφοποιήσεις αριθμών:

```python
pi = 3.14159265359
print('pi={0:.2f}'.format(pi))      # → 3.14
print('pi={0:.6f}'.format(pi))      # → 3.141593
print('f={0:>5.3f}'.format(pi))     # →   3.14
```

## **Λίστες**

Μια **λίστα (list)** αποτελείται από οντότητες χωρισμένες με κόμματα, μέσα σε αγκύλες `[ ]`.
Η λίστα είναι **μεταλλάξιμος τύπος**.

---

### Παραδείγματα

```python
Lst = []                # κενή λίστα
Lst = list()            # κενή λίστα
Lst = [1, 'Python', 3.14]  # Λίστα με τρία στοιχεία
Lst = list('Python')    # μετατροπή συμβολοσειράς σε λίστα
Lst = ['a', 'b', 'c'] + ['x', 'y', 'n']  # συγχώνευση λιστών
```

📌 Η προσπέλαση των επιμέρους στοιχείων μιας λίστας γίνεται μέσω αριθμοδείκτη. Η αρίθμηση ξεκινάει από το 0.

```python
print(Lst[0])         # εμφανίζει το 1ο στοιχείο της λίστας Lst
Lst[2] = 5            # αντικαθιστά το 3ο στοιχείο της λίστας με το 5

for i in Lst:
    print(i)          # Εμφανίζει ένα προς ένα όλα τα στοιχεία της Lst
```

---

### Λίστες πολλών επιπέδων (διαστάσεων)

Μια λίστα μπορεί να είναι **πολυδιάστατη**, δηλαδή να περιέχει άλλες λίστες.

```python
Lst2d = [[2,5,5,6,8], [9,10,7.5,4]]
```

📌 Η προσπέλαση στα επιμέρους στοιχεία μιας λίστας 2Δ γίνεται μέσω δύο αριθμοδεικτών. Η αρίθμηση κάθε επιπέδου ξεκινάει από το 0:

```python
print(Lst2d[0][0])    # → 2
print(Lst2d[1][3])    # → 4
```

## **Πίνακας: Μέθοδοι που εφαρμόζονται σε λίστες**

| **Μέθοδος** | **Περιγραφή** |

## **Συμπερίληψη λίστας**

Η **συμπερίληψη λίστας (list comprehension)** είναι ένας εξειδικευμένος τρόπος δημιουργίας μιας νέας λίστας από μια υπάρχουσα επαναληπτική οντότητα.

Lst1 = [1, 2, 3, 4, 5, 6, 7, 8]

📌 Επιστρέφει μια νέα λίστα `Lst2` με στοιχεία τα **τετράγωνα** των στοιχείων της `Lst1`:

Lst2 = [x**2 for x in Lst1]    # → [1, 4, 9, 16, 25, 36, 49, 64]
```

Lst3 = [x for x in Lst1 if x % 3 == 0]    # → [3, 6]
```

| `update(dict)` | Εισάγει στο λεξικό τα στοιχεία του λεξικού `dict`. Αν υπάρχουν ίδια κλειδιά, οι τιμές τους αντικαθίστανται από ενημερωμένες τιμές του `dict`. |

## **Bytes και bytearray**

Ένα αντικείμενο **`bytes`** αποτελείται από μία **ακολουθία byte**.
Με ένα byte μπορεί να παρασταθεί ένας αριθμός από το **0 (00000000)** μέχρι το **255 (11111111)**.
Μπορούμε να μετατρέψουμε μια συμβολοσειρά σε αντικείμενο `bytes` με κατάλληλη κωδικοποίηση.

```python
s1 = 'Λέσβος'
b1 = s1.encode('utf-8')
print(b1)
```

📌 Στο αντικείμενο `bytes` **b1** ανατίθεται η **κωδικοποιημένη** μορφή της συμβολοσειράς s1:

```python
b1  # → b'\xce\x9b\xce\xad\xcf\x83\xce\xb2\xce\xbf\xcf\x82'
```

```python
s2 = b1.decode()
print(s2)
```

📌 Στην s2 ανατίθεται η **αποκωδικοποιημένη** μορφή των χαρακτήρων του αντικειμένου `bytes` **b1**
🔤 Εκτύπωση: **Λέσβος**

---

Τα αντικείμενα **`bytearray`** περιέχουν και αυτά ακολουθίες από byte, με τη διαφορά ότι **είναι μεταλλάξιμα**, δηλαδή μπορούμε να αλλάξουμε τις τιμές των επιμέρους byte, να διαγράψουμε, να προσθέσουμε κ.λπ.
Η δομή τους μοιάζει ακριβώς με τα αντικείμενα **λίστας**.

```python
ba1 = bytearray(2)
ba2 = bytearray(b'Nikos')
ba2[1] = 97
print(ba2)
```

📌 Στη μεταβλητή **ba1** αποθηκεύονται δύο κενά byte.
📌 Στη μεταβλητή **ba2** αποθηκεύεται η λέξη `'Nikos'` σε byte μορφή.
📌 Με την εντολή `ba2[1] = 97` αντικαθίσταται το **δεύτερο byte** της **ba2** με το byte που αντιστοιχεί στο **γράμμα 'a'**, με αποτέλεσμα η νέα μορφή να είναι **b'Na...'**

| **Μέθοδος** | **Περιγραφή** |

## **Κλάσεις και αντικείμενα**

Ο ορισμός μιας κλάσης γίνεται ως εξής:

```python
class Όνομα_κλάσης:
    Ορισμοί μεταβλητών
    Ορισμοί συναρτήσεων (μεθόδων)
```

📌 Τα παραπάνω αποτελούν τα **μέλη της κλάσης**.

---

### Παράδειγμα κλάσης

```python
class proion:
    def __init__(self, p, t):
        self.perigrafi = p
        self.timi = t

    def stoixeia(self):
        print(self.perigrafi, 'Τιμή:', self.timi)

    def kostos(self, temaxia = 1):
        return self.timi * temaxia
```

---

Το ακόλουθο τμήμα κώδικα δημιουργεί δύο αντικείμενα της κλάσης `proion` και αναθέτει στις μεταβλητές `p1` και `p2`.
Κατά τη δημιουργία τους, ανατίθενται οι τιμές των ορισμάτων στη **μεταβλητή περιγραφή** και την **τιμή** του καθενός:

```python
p1 = proion('πουκάμισο', 45)
p2 = proion('ρολόι', 125)

p1.stoixeia()     # εμφανίζει τα στοιχεία του αντικειμένου p1
p2.timi = 100     # αναθέτει νέα τιμή του αντικειμένου p2
print(p2.kostos(5))  # εμφανίζει το κόστος για 5 τεμάχια
```

---

📌 Η **προσπέλαση** των επιμέρους μελών (μεταβλητών ή μεθόδων) ενός αντικειμένου γίνεται με **χρήση του τελεστή τελείας `.`**

## **Υπερφόρτωση τελεστών**

Η **υπερφόρτωση τελεστών** μας δίνει τη δυνατότητα να **επαναορίσουμε** τη λειτουργία ενός τελεστή όταν αυτός εφαρμόζεται σε **αντικείμενα κλάσεων**.

📌 Η υπερφόρτωση τελεστών υλοποιείται μέσω της **ειδικής μεθόδου** `__όνομα__`, η οποία καλείται **αυτόματα** όταν χρησιμοποιείται ο αντίστοιχος τελεστής.

📌 Για κάθε τελεστή που μπορούμε να επαναορίσουμε με χρήση των "μαγικών μεθόδων" της Python, ακολουθεί ο πίνακας υπερφόρτωσης τελεστών.

---

## **Πίνακας Υπερφόρτωσης Τελεστών στην Python**

| **Λειτουργία** | **Τελεστής** | **Μέθοδος υπερφόρτωσης** |
| -------------- | ------------ | ------------------------ |
| Πρόσθεση | `+` | `__add__()` |
| Αφαίρεση | `-` | `__sub__()` |
| Πολλαπλασιασμός | `*` | `__mul__()` |
| Ύψωση σε δύναμη | `**` | `__pow__()` |
| Διαίρεση | `/` | `__truediv__()` |
| Υπολοίπου ακέραιας διαίρεσης | `%` | `__mod__()` |
| Μικρότερο | `<` | `__lt__()` |
| Μικρότερο ή ίσο | `<=` | `__le__()` |
| Μεγαλύτερο | `>` | `__gt__()` |
| Μεγαλύτερο ή ίσο | `>=` | `__ge__()` |
| Ίσο | `==` | `__eq__()` |
| Διάφορο | `!=` | `__ne__()` |

---

### Παράδειγμα υπερφόρτωσης τελεστή `+`

```python
def __add__(self, o):
    if self.perigrafi == o.perigrafi:
        t = proion(self.perigrafi, (self.timi + o.timi) / 2)
        return t
    else:
        print('Δεν προσθέτω ανόμοια προϊόντα!')
        return None
```

📌 **Υπερφόρτωση του τελεστή `+` για την κλάση `proion`.**

---

### Παράδειγμα υπερφόρτωσης τελεστή `>`

```python
def __gt__(self, o):
    if self.timi > o.timi:
        return True
    else:
        return False
```

📌 **Υπερφόρτωση του τελεστή `>` για την κλάση `proion`.**

## **Κληρονομικότητα**

Με την **κληρονομικότητα** μπορούν να παραχθούν **νέες κλάσεις (παράγωγες)** από μία **υπάρχουσα κλάση (βασική)**.

Για να ορίσουμε μια κλάση η οποία θα βασίζεται σε μία άλλη κλάση, μετά το όνομα της νέας κλάσης τοποθετούμε μέσα σε παρενθέσεις το όνομα της βασικής κλάσης:

```python
class παραγόμενη_κλάση(βασική_κλάση):
    σώμα παραγόμενης κλάσης
```

Η νέα παραγόμενη κλάση **κληρονομεί όλα τα χαρακτηριστικά** (μεταβλητές και μεθόδους) της βασικής κλάσης.

---

### Παράδειγμα

```python
class A:
    def __init__(self, n):
        self.name_a = n
        print('Μέθοδος δόμησης κλάσης A —', self.name_a)

    def meth_a(self):
        print('Μέθοδος βασικής κλάσης A —', self.name_a)
```

```python
class B(A):   # Ορίζεται μια νέα κλάση B, παράγωγη της κλάσης A
    def meth_b(self):
        print('Μέθοδος παραγόμενης κλάσης B')
```

## **Εξαιρέσεις (Exceptions)**

Οι **εξαιρέσεις (exceptions)** είναι **σφάλματα** τα οποία μπορεί να προκύψουν κατά την εκτέλεση ενός προγράμματος.

Η διαχείρισή τους γίνεται με τις εντολές `try` και `except`.

---

### Γενική σύνταξη

```python
try:
    # κώδικας που μπορεί να προκαλέσει εξαίρεση
except ΌνομαΚλάσηςΕξαίρεσης-1:
    # κώδικας χειρισμού εξαίρεσης 1
except ΌνομαΚλάσηςΕξαίρεσης-2:
    # κώδικας χειρισμού εξαίρεσης 2
...
except:
    # κώδικας χειρισμού για όλες τις υπόλοιπες εξαιρέσεις
```

---

### Παράδειγμα:

```python
try:
    a = int(input('Δώσε έναν αριθμό: '))
    b = int(input('Δώσε και άλλον αριθμό: '))
    print(a / b)
    if a > b:
        c = a - b
        print(c)
```

📌 **Στο μπλοκ `try` μπαίνει ο κώδικας που μπορεί να προκαλέσει εξαίρεση.**

---

```python
except NameError as e1:
    print('Err… κάτι ξέχασες!')
    print(e1)
```

📌 **Συλλαμβάνει εξαίρεση τύπου `NameError`.**
