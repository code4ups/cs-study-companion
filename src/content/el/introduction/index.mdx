        <h1 className="text-3xl font-bold text-indigo-700 mb-6">Εισαγωγή</h1>

        Το μάθημα Επιστήμης Υπολογιστών του Διπλώματος (DP) έχει σχεδιαστεί για μαθητές ηλικίας 16 έως 19 ετών. Το πρόγραμμα σπουδών στοχεύει να εξετάσει βασικές έννοιες της επιστήμης υπολογιστών: βασικές αρχές υπολογιστών, δίκτυα, βάσεις δεδομένων και μηχανική μάθηση, και στη συνέχεια να εφαρμόσει πρακτικές δεξιότητες για την υποστήριξη της διαδικασίας υπολογιστικής σκέψης με σκοπό την επίλυση προβλημάτων. Όπως συμβαίνει με όλα τα στοιχεία του DP, το μάθημα αυτό καλλιεργεί τα χαρακτηριστικά του προφίλ μαθητή του IB (βλ. σελίδα vi) στα μέλη της σχολικής κοινότητας.

        <h2 className="text-2xl font-semibold mt-8 mb-3">Πώς να χρησιμοποιήσετε αυτό το βιβλίο</h2>

        Ο στόχος αυτού του βιβλίου είναι να αναπτύξει την εννοιολογική κατανόηση, να βοηθήσει στην ανάπτυξη δεξιοτήτων και να προσφέρει ευκαιρίες για εδραίωση της γνώσης και της κατανόησης μέσω της εξάσκησης.

        Τα πλαίσια και οι ενότητες που υπάρχουν σε όλο το βιβλίο έχουν σχεδιαστεί για να υποστηρίζουν αυτούς τους στόχους, επισημαίνοντας περιεχόμενο που σχετίζεται με συγκεκριμένες ιδέες και έννοιες, καθώς και ευκαιρίες για εξάσκηση. Ακολουθεί μια επισκόπηση αυτών των χαρακτηριστικών:

        <h3 className="text-xl font-semibold mt-6 mb-2">Ανάπτυξη εννοιολογικής κατανόησης</h3>

        **Καθοδηγητικές ερωτήσεις**
        Κάθε θέμα ξεκινά με μια καθοδηγητική ερώτηση που σας παρακινεί να σκεφτείτε. Όταν ξεκινήσετε να μελετάτε ένα θέμα, μπορεί να μην είστε σε θέση να απαντήσετε σε αυτές τις ερωτήσεις με αυτοπεποίθηση ή πληρότητα, αλλά με τη μελέτη του θέματος θα μπορείτε να τις απαντάτε με ολοένα και μεγαλύτερο βάθος. Επομένως, θα πρέπει να τις λαμβάνετε υπόψη καθώς προχωράτε στο θέμα και να επιστρέφετε σε αυτές όταν επανεξετάζετε την κατανόησή σας.

        **Συνδετικές ερωτήσεις**
        Οι συνδετικές ερωτήσεις μέσα σε κάθε θέμα αναδεικνύουν τις συνδέσεις μεταξύ του περιεχομένου που συζητείται εκεί και άλλων μερών του μαθήματος.

        **Θεωρία της γνώσης (TOK)**
        Αυτό αποτελεί σημαντικό μέρος του Προγράμματος Διπλώματος IB. Επικεντρώνεται στην κριτική σκέψη και στην κατανόηση του πώς αποκτούμε τη γνώση μας για τον κόσμο. Τα στοιχεία του TOK σε αυτό το βιβλίο θέτουν ερωτήσεις που αναδεικνύουν αυτά τα ζητήματα.

        Τμήματα του βιβλίου έχουν μια χρωματιστή μπάρα στην άκρη της σελίδας ή δίπλα σε μια ερώτηση. Αυτό υποδηλώνει ότι το υλικό απευθύνεται σε μαθητές που σπουδάζουν το μάθημα επιστήμης υπολογιστών του DP σε ανώτερο επίπεδο. Ο όρος AHL σημαίνει «πρόσθετο ανώτερο επίπεδο».

        Τα στοιχεία συνδέσμων στο περιθώριο θα σας κατευθύνουν σε άλλα μέρη του βιβλίου όπου μια έννοια εξετάζεται περαιτέρω ή σε διαφορετικό πλαίσιο. Μπορεί επίσης να σας παραπέμψουν σε προϋπάρχουσα γνώση ή δεξιότητα που θα χρειαστείτε, ή να σας δώσουν έναν διαφορετικό τρόπο να σκεφτείτε κάτι.

        <h3 className="text-xl font-semibold mt-6 mb-2">Ανάπτυξη δεξιοτήτων</h3>

        **ATL – Προσεγγίσεις στη μάθηση**
        Αυτά τα στοιχεία για τις προσεγγίσεις στη μάθηση (ATL) σας ενθαρρύνουν να αναπτύξετε στρατηγικές που υποστηρίζουν τις δεξιότητες ATL στην επικοινωνία, την αυτοδιαχείριση, την έρευνα, τη σκέψη και τις κοινωνικές δεξιότητες.

        **Πρακτικές δεξιότητες**
        Οι βασικές πρακτικές δεξιότητες στην επιστήμη υπολογιστών είναι η αλγοριθμική σκέψη και ο προγραμματισμός. Αυτά τα πλαίσια επισημαίνουν ευκαιρίες για την ανάπτυξη και εφαρμογή των πρακτικών σας δεξιοτήτων στο πλαίσιο της διαδικασίας υπολογιστικής σκέψης.

        **Αυτό το πρόβλημα στην πράξη (TPIP)**
        Αυτό το στοιχείο εξετάζει παραδείγματα εφαρμογής της επιστήμης υπολογιστών στην επίλυση πραγματικών προβλημάτων.

        **Εσωτερική και εξωτερική αξιολόγηση**
        Αυτές οι ενότητες του βιβλίου παρέχουν καθοδήγηση για το πώς να εφαρμόσετε τη διαδικασία υπολογιστικής σκέψης στη δική σας υπολογιστική λύση, που αποτελεί την εσωτερική αξιολόγηση για την επιστήμη υπολογιστών, καθώς και για το πώς να προετοιμαστείτε για τις εξετάσεις Paper 1 και Paper 2.

        **Βασικός όρος**
        Αυτά παρουσιάζουν τους ορισμούς σημαντικής ορολογίας που χρησιμοποιείται στην επιστήμη υπολογιστών.

        <h3 className="text-xl font-semibold mt-6 mb-2">Εξάσκηση</h3>

        **Λυμένα παραδείγματα**
        Πρόκειται για παραδείγματα βήμα-βήμα σχετικά με το πώς να απαντάτε σε ερωτήσεις. Θα πρέπει να τα εξετάζετε προσεκτικά, κατά προτίμηση αφού πρώτα επιχειρήσετε να απαντήσετε στην ερώτηση μόνοι σας.

        **Ερωτήσεις εξάσκησης**
        Αυτές έχουν σχεδιαστεί για να σας προσφέρουν επιπλέον εξάσκηση στη χρήση των γνώσεών σας και να σας επιτρέψουν να ελέγξετε τη δική σας κατανόηση και πρόοδο.

        **Δραστηριότητα**
        Αυτές σας δίνουν την ευκαιρία να εφαρμόσετε τις γνώσεις και τις δεξιότητές σας, συχνά με πρακτικό τρόπο.

        **Ερωτήσεις τέλους ενότητας**
        Χρησιμοποιήστε αυτές τις ερωτήσεις στο τέλος κάθε ενότητας για να συνδυάσετε έννοιες από την ενότητα και να εξασκηθείτε στην απάντηση ερωτήσεων τύπου εξετάσεων.
