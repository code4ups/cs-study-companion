    # A3 – Ερωτήσεις τέλους ενότητας

    ### Πόσο καλά κατανοείς τις αρχές, τις δομές και τις λειτουργίες που διέπουν τις βάσεις δεδομένων;

    ---

    ## Ανασκόπηση ύλης

    1. Χρησιμοποιώντας τις γνώσεις σου από αυτή την ενότητα, A3, απάντησε όσο το δυνατόν πληρέστερα στην κατευθυντήρια ερώτηση:
    **Ποιοι είναι οι κανόνες, οι δομές και οι λειτουργίες που αποτελούν τη βάση των συστημάτων βάσεων δεδομένων;**
    *[6 μονάδες]*

    ---

    ## Ερωτήσεις τύπου εξετάσεων

    2. **Περιέγραψε μία ανωμαλία που θα μπορούσε να εμφανιστεί σε μία μη κανονικοποιημένη βάση δεδομένων.**
    *[2 μονάδες]*

    3. **Περιέγραψε δύο χαρακτηριστικά των δεδομένων σε μια αποθήκη δεδομένων.**
    *[4 μονάδες]*

    4. **Εξήγησε ένα εργαλείο που θα μπορούσε να χρησιμοποιηθεί για τη λήψη αποφάσεων σε μία αποθήκη δεδομένων.**
    *[3 μονάδες]*

    5. Σκέψου την παρακάτω βάση δεδομένων που αποθηκεύει δεδομένα σχετικά με πελάτες και τις αγορές τους σε ένα ηλεκτρονικό κατάστημα.

    **a. Κατασκεύασε μία εντολή SQL που θα εμφανίζει όλους τους πελάτες που έχουν επισκεφθεί την Ιταλία.**
    *[3 μονάδες]*

    **b. Χρησιμοποιώντας τα δεδομένα της προβολής, κανονικοποίησε τη βάση δεδομένων μέχρι την τρίτη κανονική μορφή.**
    *[5 μονάδες]*

    6. **Εξήγησε γιατί μια βάση δεδομένων μπορεί να χρησιμοποιεί όψεις (views) όταν προβάλλει δεδομένα στους χρήστες.**
    *[3 μονάδες]*

    7. **Περιέγραψε τον ρόλο των ιδιοτήτων ACID κατά την ενημέρωση μιας εγγραφής σε μία βάση δεδομένων.**
    *[4 μονάδες]*

    8. **Εξήγησε γιατί οι χωρικές βάσεις δεδομένων χρησιμοποιούνται για την αποθήκευση δεδομένων με γεωγραφική πληροφορία.**
    *[3 μονάδες]*

    9. **Προσδιόρισε τον σκοπό της συνάρτησης συγκέντρωσης MAX.**
    *[2 μονάδες]*

    10. **Παρουσίασε δύο μορφές ασφάλειας δεδομένων σε μία κατανεμημένη βάση δεδομένων.**
    *[4 μονάδες]*

    11. **Εξήγησε πώς η ανακάλυψη συσχετιστικών κανόνων μπορεί να χρησιμοποιηθεί ώστε να ενθαρρυνθούν οι καταναλωτές να αγοράσουν περισσότερα προϊόντα.**
    *[5 μονάδες]*
